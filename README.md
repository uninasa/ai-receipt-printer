## 🛠️ 快速开始

### 1. 下载安装

- 下载最新版本的 `Release` 文件夹
- 双击运行 `ai_receipt_reader.exe`

### 2. 配置 API Key

- 点击右上角设置图标
- 在 [DeepSeek 平台](https://platform.deepseek.com) 获取免费 API Key
- 输入并保存

### 3. 一键生成

```
选择项目目录 → 选择对话文件 → 勾选问答 → 点击"究极 Agent 赋能" → 完成！
```

详细使用说明请查看 [📖 操作手册](./MANUAL.md)
